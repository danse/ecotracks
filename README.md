### Ecotracks

Experimental project, visualising data like in [this beautiful New
York Times
article](http://www.nytimes.com/interactive/2013/10/09/us/yellen-fed-chart.html),
but for all world's countries

#### Running

    cd data/
    make

Requires haskell's cabal. Then spin a server like in

    python -m http.serve 8080
